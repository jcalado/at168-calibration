# AT-D168UV Calibration GUI

This is a fork of [DualTachyon/at168-calibration](https://github.com/DualTachyon/at168-calibration) that adds a graphical user interface for backing up and restoring calibration data.

Please follow DualTachyon's recommendations and documentation for calibration procedures and device handling.

## Support the original author

* If you like this work, please support DualTachyon through [Ko-Fi](https://ko-fi.com/DualTachyon)

## About

With this tool, you can easily backup the calibration data and device information to files. With these 2 files, the AT-D168UV is virtually unbrickable and you will always be able to fully recover a unit with bad flash contents, as if it came out from the factory.

This will be most useful for the upcoming [REFW-168](https://refw-168.duckdns.org).
