# Backing up AnyTone AT-D168UV calibration data and device info.
With this script, you can easily backup the calibration data and device information to files. With these 2 files, the AT-D168UV is virtually unbrickable and you will always be able to fully recover a unit with bad flash contents, as if it came out from the factory.

This will be most useful for the upcoming [REFW-168](https://refw-168.duckdns.org).
